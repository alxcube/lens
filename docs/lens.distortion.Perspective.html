new Perspective(reverseMatrix, denominator)

Perspective Distortion (a ratio of affine distortions).

Source:
  • distortion/perspective.js, line 56
Implements:
  • DistortionInterface
Mixes In:
  • lens.mixins.createsBestFitViewportFromApexes
See:
  • Perspective distortion details at ImageMagick docs
  • Perspective distortion at ImageMagick source

    p(x,y)    c0*x + c1*y + c2
u = ------ = ------------------
    r(x,y)    c6*x + c7*y + 1

    q(x,y)    c3*x + c4*y + c5
v = ------ = ------------------
     r(x,y)    c6*x + c7*y + 1

denominator = Sign of 'r', or the denominator affine, for the actual image.
This determines what part of the distorted image is 'ground'
side of the horizon, the other part is 'sky' or invalid.
Valid values are  +1.0  or  -1.0  only.
Parameters:
Name Type Description
reverseMatrix Array.<number>

Perspective projection matrix for reverse pixel mapping.
denominator number

Denominator for mapping validity calculation.
